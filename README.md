Data extraida con un script personalizado, utilizado en ninjatrader para capturar el ADN completo de una vela
en temporalidad de un minuto. Son mas de 80 columnas con datos que pueden ser utiles para el analisis post-mortem
de una operacion.
Data limpia y revisada para su uso en Backtesting de estrategias en el instrumento NQ. En total 6 años de historia
muy valiosos para estudio.
El original contiene mas de 2 millones de velas.
